# AgentGen

A framework for creating code agents.

## Installation

```bash
# Install with pip
pip install -e .

# Or install with uv
uv pip install -e .
```

## Features

- Agent creation and management
- CLI tools
- Extensible architecture

## Usage

```bash
# Check the version
agentgen --version

# Run the CLI
agentgen
```

## Development

```bash
pip install -e ".[dev]"
```
